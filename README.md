# Backoffice Viajes Altairis

Demo MVP comercial hecho con .NET y Next.js.
Gestiona catálogo de hoteles, disponibilidad e información de reservas.

## Instrucciones de despliegue

1. Clonar el repositorio
2. Ejecutar: 
```sh
docker-compose up --build
```
3. Acceder a:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:5000
   - Base de datos SQL Server: localhost:1433

## Demo de datos

Esos datos que se introducen por primera vez del despliegue están en [`backend/Infra/Data/SeedData.cs`](backend/Infra/Data/SeedData.cs)
